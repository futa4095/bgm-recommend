# bgm-recommend

Helping you find today's background music

## Installation

```sh
npm install bgm-recommend
```

## Usage

Set the API Key to an environment variable and execute the command.

```sh
export BGM_RECOMMEND_API_KEY=YOUR_API_KEY
npx bgm-recommend
```
